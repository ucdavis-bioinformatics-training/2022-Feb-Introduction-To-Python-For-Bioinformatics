# Putting it all together

Now that you've learned the basics of programming, it's time to write some beginning code. Let's write a program that will print the count for each nucleotide in a DNA sequence. You will need to use the "list" function which, when applied to a string, will return the string as a list of characters. You will also need to use a loop and a dictionary. You can use this sequence to begin with:

**CGGTAGTCGAGCTGCGGATATAATATGCATATAGATCGCACGCTAGCTCATAAAAGCATGCATGCGGCTAGCTGCTGATCGTGTCG**
